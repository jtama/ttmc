Qu’est-ce que NaN en Java ?

Not a Number, il s'agit d'une valeur à virgule flottante spéciale pour signaler les dépassements de capacité et les erreurs. Elle est générée lorsqu'un nombre à virgule flottante est divisé par zéro ou lorsque la racine carrée d'un nombre négatif est calculée.

Pourquoi dit-on que les Enum Java sont implicitement final ?

Car le compilateur interdit d'étendre une enum. Le bytecode les marque comme final, pour garantir leur unicité (singleton-like)

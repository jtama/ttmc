Que fait la propriété background-color ?

Elle définit la couleur de fond d’un élément.

À quoi servent les media queries en CSS ?

Elles permettent d’appliquer des styles différents selon la taille de l’écran, utiles pour le responsive design.

Quelle est la différence entre id et class en CSS ?

Un id est unique par page (#id), une class peut être utilisée plusieurs fois (.class).

Quelle est la différence entre relative, absolute et fixed en CSS ?

  • relative : déplacement par rapport à sa position initiale.
  • absolute : relatif au parent positionné.
  • fixed : relatif à la fenêtre (viewport).

Quelle est la différence entre rem, em et px ?

  • px : pixels fixes
  • em : relatif à la taille de la police du parent.
  • rem : relatif à la taille de la police à la racine (html)

Que signifie la propriété box-sizing: border-box ?

Le padding et la border sont inclus dans la taille (width et height).

Quelle est la différence entre relative, absolute, fixed et sticky en CSS ?

  • relative : positionne l'élément par rapport à sa position normale.
  • absolute : positionne par rapport au parent positionné le plus proche.
  • fixed : positionne par rapport à la fenêtre du navigateur (reste fixe au scroll).
  • sticky : combine relative et fixed, devient fixe à partir d’un seuil de scroll.

Comment fonctionne le système de spécificité CSS et dans quel cas une règle est prioritaire sur une autre ?

Du plus prioritaire au moins prioritaire :

  1. Styles en ligne (inline)
  2. !important
  3. Sélecteurs avec ID
  4. Sélecteurs de classes, attributs et pseudo-classes
  5. Sélecteurs de balises et pseudo-éléments.

Comment fonctionne le contain et le will-change en CSS et pourquoi sont-ils utilisés ?

Ce sont des optimisations avancées de rendu :

  • contain limite la zone d’invalidation et améliore la performance du rendu.
  • will-change avertit le navigateur d’un futur changement (ex : transform, opacity) pour optimiser le GPU.

Quelles sont les conditions pour qu’un élément soit formaté en block formatting context (BFC) et à quoi servent-elles ?

Un BFC est une zone d’isolation visuelle.

  • Il empêche les éléments flottants (floats) de sortir de leur conteneur, entre autres.
  • Un BFC est déclenché par : overflow: hidden|auto|scroll, `display: flow-root, float, etc."
